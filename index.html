О нас

Приветствую!

Меня зовут Ольга Воробьёва — руководитель компании «ОВАпарт». Более 3 лет помогаю подобрать и забронировать комфортные апартаменты для краткосрочного проживания в престижных районах Москвы.

Наши апартаменты подойдут:

  • для командировок (предоставляем отчётные документы)
  • для туристов (рядом метро и достопримечательности)
  • для студентов, спортсменов и участников мероприятий
  • для размещения гостей и родственников
  • для иностранных гостей (общаемся на английском)
